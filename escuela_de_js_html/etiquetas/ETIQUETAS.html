Etiquetas

en html todo se caracteriza por un elemento de apertura y un elemento de cierre por ejemplo:

  • apertura<*>
  • cierre < /* >

tambien podemos agrear atributos y valores

ejemplo:
  • < h1 class = < title >
  • en este caso class seria un atributo y title un valor

los atributos nos van a servir para usar cosas, en este caso en particular tenemos class que es uno de los mas usados y nos va a servir para cominicarnos con CSS

Etiquetas y sus clases

    LAYOUT: es el diseño de toda la pagina entre ella tenemos las siguientes etiquetas.

  • Header
  • nabyar
  • section
  • article
  • aside
  • footer

    • ENLACES: sirve para poner distintos enlaces para que el usuario sea redirigido a otra pagina web. para este unicamete usamos la etiqueta

  • .a

    • TEXTOS: la importancia es vital y el buen manejo de sus etiquetas para una estructura entendible, las etiquetas que tenemos son

  • h1....h6 - esta es la importancia que tiene cada texto desde h1 hasta h6 esto es importante para construir un sitio web que tenga sentido.
  • p: indica el inicio y el final de un parrafo
  • span: con este podemos destacar texto ya sea curveandolo, subrayandolo o poniendolo en negrita

    • IMAGENES Y VIDEOSsu propio nombre lo indica, las etiquetas que usamos son:

  • img
  • svg
  • iframe
  • video

    • FORMULARIOS es importanticimo y lo vamos a encontrar mucho en programacion frontend

  • .from: la estructura inicial del formuario
  • .input: existen varios tipos de input
  • .label: es ese texto inicial que funciona como el titulo del input
  • .button: los botones nos van a permitir enviar los formularios

    • LISTAScomo su nombre lo indica nos permite hacer listas

  • ul: esto nos va a ayudar
  • li: a crear listas ordenadas o no,
  • li: donde van a ir los distintso items
